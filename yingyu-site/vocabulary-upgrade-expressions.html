别再说"very good"了：升级你的表达词汇库

发布于 2026-05-12  |  分类：词汇辨析

如果你的英文表达里经常出现 very good、very bad、very big、very small……那你的词汇库需要一次升级了。"very + 形容词"不算错，但它是初学者的标配。一个精准的单词，胜过三个非常。

为什么应该减少"very"的使用

哈佛大学语言学家 Stephen Pinker 说过："Very is the weakest word in the English language."原因很简单——very 是修饰词，它本身不携带具体信息。说"very big"不如直接说"enormous"，后者在听者脑中产生的画面要清晰得多。

20 个最常用的升级替换

以下是从日常对话到职场写作都高频的替换词：

描述大小 / 程度

very big→enormous / massive
very small→tiny / miniature
very fast→rapid / swift
very slow→sluggish / gradual

描述品质

very good→excellent / superb / outstanding
very bad→terrible / awful / dreadful
very important→crucial / essential / vital
very interesting→fascinating / captivating

描述情绪

very happy→delighted / thrilled / overjoyed
very sad→heartbroken / devastated
very angry→furious / enraged
very scared→terrified / petrified
very tired→exhausted / drained
very surprised→astonished / stunned

描述外观

very beautiful→gorgeous / stunning / breathtaking
very ugly→hideous / repulsive
very clean→spotless / immaculate
very dirty→filthy / grimy
very old→ancient / antique
very new→brand-new / cutting-edge
The presentation was not just good — it was outstanding.
这个演示不只是好——简直出色。
After the marathon, I was exhausted.
跑完马拉松之后，我筋疲力尽。

怎么练习？

建议每天挑 5 个替换词，在日记或聊天中有意识地用出来。写作时写完一句含有 "very" 的话，回头看看上面的列表，把 "very + 形容词" 替换成精准词。坚持两周，你的表达会自然升级。
